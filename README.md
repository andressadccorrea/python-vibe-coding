# 🐍 Python + Vibe Coding: um experimento pessoal

> Aprendendo Python do zero enquanto investigo como a IA generativa muda (ou não) a forma de programar.

---

## Sobre o projeto

Este repositório é um diário público de aprendizado.

A ideia surgiu de uma curiosidade dupla: quero aprender Python na prática **e** entender de dentro como funciona o Vibe Coding — a prática de programar em colaboração com ferramentas de IA como o Claude ou o Copilot.

Meu perfil é de análise de dados, então no horizonte está algo maior: usar Python para construir modelos preditivos reais. Mas toda jornada começa pelo começo.

---

## Metodologia

Cada semana combina três elementos:

| Elemento | O que é |
|---|---|
| 📺 Curso em Vídeo | Base teórica em português, estruturada e progressiva |
| 🤖 Vibe Coding | Mini-desafio semanal resolvido em colaboração com IA |
| 📝 Observações | Registro crítico da experiência — o que funcionou, o que falhou |

A rotina é de segunda a sexta, com 1–2 horas por semana.

---

## Roteiro

### Fase 1 — Fundamentos (semanas 1–4)
Variáveis, condicionais, laços de repetição e primeiro projeto integrador.

| Semana | Tema | Pergunta de investigação |
|---|---|---|
| 01 | Variáveis e tipos de dados | A IA me ajudou a começar mais rápido do que eu conseguiria sozinha? |
| 02 | Condicionais | Onde o Vibe Coding economiza tempo e onde me faz depender demais? |
| 03 | Laços de repetição | O Vibe Coding muda a forma como penso o problema antes de escrever o código? |
| 04 | Síntese + primeiro conteúdo | Após um mês, o Vibe Coding me tornou mais ou menos autônoma? |

### Fase 2 — Estrutura de dados (planejada)
Listas, dicionários, funções e primeiros scripts de manipulação de dados reais.

### Fase 3 — Análise de dados (planejada)
`pandas`, `matplotlib` e exploração de datasets reais com Python.

### Fase 4 — Modelos preditivos (objetivo de longo prazo)
`scikit-learn`, regressão, classificação e validação de modelos.
